# Recommended Minimum Documentation — MPS.BR Level F (Managed)

The recommended minimum documentation for Level F. Processes: GRE, GPR, GQA, GCO and MED.

## Structure
- requirements/ — Functional, non-functional, traceability and change requests
- projects/ — Roadmap, schedule, risks, status and releases
- quality/ — Quality plan and quality report
- configuration/ — Configuration management plan and baseline
- measurement/ — Metrics
