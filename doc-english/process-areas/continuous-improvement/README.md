# 🚀 Process Area — Continuous Improvement

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in MPS.BR
Continuous process improvement and innovation through causal analysis. Maps to **Innovation and Deployment Management (GIM)**.

## 🎯 What problem it solves
Solves **stagnation**: processes that do not evolve become obsolete.

## ✅ What belongs in this area
- Root cause analysis
- Improvement plan
- Process innovation
- Improvement opportunities and history
- Benchmark and retrospectives

## 🚫 What does NOT belong in this area
- Statistical control (→ quantitative-management)
- Raw metrics (→ measurement)
