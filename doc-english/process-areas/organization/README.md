# 🏛️ Process Area — Organization

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in MPS.BR
Shared assets: standards, templates, training and lessons learned. Supports **Organizational Process Definition (DFP)** and **HR Management (GRH)**.

## 🎯 What problem it solves
Solves **knowledge that is lost and not reused**.

## ✅ What belongs in this area
- Development/documentation standards
- Organizational templates
- Training
- Lessons learned

## 🚫 What does NOT belong in this area
- Operational processes (→ processes)
- Metrics (→ measurement)
