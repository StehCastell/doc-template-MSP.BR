# 🧭 Process Area — Decisions

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in MPS.BR
Analysis and recording of decisions with criteria. Maps to **Decision Management (GDE)**.

## 🎯 What problem it solves
Solves **important decisions made without criteria or record**.

## ✅ What belongs in this area
- Decision criteria
- Alternative analysis
- Decision log

## 🚫 What does NOT belong in this area
- Product architecture decisions (→ architecture)
- Risks (→ risk-management)
