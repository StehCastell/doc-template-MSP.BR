# 📈 Process Area — Quantitative Management

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in MPS.BR
Statistical management of process and project. Maps to **Quantitative Project Management (GQP)** and **Organizational Process Performance (DEP)**.

## 🎯 What problem it solves
Solves **not knowing whether targets are statistically achievable**.

## ✅ What belongs in this area
- Quantitative objectives
- Performance baselines
- Statistical process control
- Prediction models

## 🚫 What does NOT belong in this area
- Raw metric collection (→ measurement)
- Process improvement (→ continuous-improvement)
