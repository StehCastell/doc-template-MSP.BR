# 📊 Process Area — Measurement

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in MPS.BR
Data collection and analysis for decisions. Maps to **Measurement (MED)**.

## 🎯 What problem it solves
Solves **deciding by guesswork**.

## ✅ What belongs in this area
- Metric definition
- Project/process indicators
- Metrics baseline
- Dashboards and reports

## 🚫 What does NOT belong in this area
- Individual defects (→ quality)
- Schedule (→ projects)
