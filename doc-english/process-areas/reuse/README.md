# ♻️ Process Area — Reuse

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in MPS.BR
Management of reusable assets (components, libraries, patterns). Maps to **Reuse Management (GRU)**.

## 🎯 What problem it solves
Solves the **rework of rebuilding what already exists**.

## ✅ What belongs in this area
- Reuse criteria
- Component catalog/library
- Reusable assets

## 🚫 What does NOT belong in this area
- General organizational standards (→ organization)
- Architecture (→ architecture)
