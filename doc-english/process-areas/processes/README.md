# ⚙️ Process Area — Processes

> This folder is **self-contained**: everything you need to understand the area is here.

## 📌 What this area is in MPS.BR
How work is done: development, change, testing and release. Supports **Process Definition** and **Process Assessment and Improvement (AMP)**.

## 🎯 What problem it solves
Solves **'everyone their own way'**.

## ✅ What belongs in this area
- Development process
- Requirements process
- Testing process
- Change/release process

## 🚫 What does NOT belong in this area
- Standards and templates (→ organization)
- Metrics (→ measurement)
