# Functional Requirements

## Objective

Document the features that shall be implemented by the system.

---

## Requirements

### FR-001

**Name:** User Authentication

**Description:**  
The system must allow authenticated users to access it using valid credentials.

**Priority:** High

**Acceptance Criteria:**

- [ ] Allow login with valid credentials;
- [ ] Prevent access with invalid credentials;
- [ ] Log authentication attempts.

---

### FR-002

**Name:** Password Recovery

**Description:**  
The system must allow password reset upon validation of the user's identity.

**Priority:** Medium

**Acceptance Criteria:**

- [ ] Allow reset request;
- [ ] Send instructions to the user;
- [ ] Log the operation.
