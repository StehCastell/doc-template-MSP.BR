# Reuse Criteria

## Objective

Define criteria for identifying and approving reusable assets.

---

## Criteria

* Documentation available;
* Tests performed;
* Proven usage;
* Ease of maintenance;
* Technological compatibility.

---

## Evaluation Process

1. Asset identification.
2. Criteria evaluation.
3. Approval.
4. Inclusion in the catalog.
