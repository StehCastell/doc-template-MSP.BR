# User Stories

## Objective

Record user needs in a simple, value-oriented way.

---

## Stories

### US-001

**As a** customer  
**I want** to recover my password  
**So that** I can access my account if I forget it.

#### Acceptance Criteria

- [ ] Allow request by email.
- [ ] Send reset instructions.
- [ ] Display a confirmation message.
