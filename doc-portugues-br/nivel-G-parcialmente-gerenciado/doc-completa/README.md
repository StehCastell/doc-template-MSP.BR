# Documentação Completa — MPS.BR Nível G (Parcialmente Gerenciado)

Conjunto **completo** de documentos do Nível G. Processos: Gerência de Requisitos (GRE) e Gerência de Projetos (GPR).

## Estrutura
- **requisitos/** — Requisitos funcionais, não funcionais, casos de uso, rastreabilidade e solicitações de mudança
- **projetos/** — Roteiro, cronograma, riscos, status, lançamentos e lições aprendidas
