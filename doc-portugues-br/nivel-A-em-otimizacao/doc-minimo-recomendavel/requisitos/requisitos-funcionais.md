# Requisitos Funcionais

## Objetivo

Documentar as funcionalidades que deverão ser implementadas pelo sistema.

---

## Requisitos

### RF-001

**Nome:** Autenticação de Usuários

**Descrição:**  
O sistema deve permitir que usuários autenticados realizem acesso por meio de credenciais válidas.

**Prioridade:** Alta

**Critérios de Aceitação:**

- [ ] Permitir login com credenciais válidas;
- [ ] Impedir acesso com credenciais inválidas;
- [ ] Registrar tentativas de autenticação.

---

### RF-002

**Nome:** Recuperação de Senha

**Descrição:**  
O sistema deve permitir a redefinição de senha mediante validação da identidade do usuário.

**Prioridade:** Média

**Critérios de Aceitação:**

- [ ] Permitir solicitação de redefinição;
- [ ] Enviar instruções ao usuário;
- [ ] Registrar a operação.
