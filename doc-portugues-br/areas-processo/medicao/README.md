# 📊 Área de Processo — Medição

> Esta pasta é **autossuficiente**: tudo o que você precisa para entender a área está aqui.

## 📌 O que é esta área no MPS.BR
Coleta e análise de dados para decisão. Corresponde ao processo **Medição (MED)**.

## 🎯 Qual problema ela resolve
Resolve o **decidir no achismo**.

## ✅ O que entra nesta área
- Definição de métricas
- Indicadores de projeto/processo
- Baseline de métricas
- Dashboards e relatórios

## 🚫 O que NÃO entra nesta área
- Defeitos individuais (→ qualidade)
- Cronograma (→ projetos)

## 📈 Como evolui do nível G ao A

| Nível MPS.BR | Evolução da área |
|---|---|
| Nível G — Parcialmente Gerenciado | — (ainda não aplicável) |
| Nível F — Gerenciado | Métricas básicas coletadas. |
| Nível E — Parcialmente Definido | Conjunto de métricas padronizado. |
| Nível D — Largamente Definido | Medição integrada ao processo. |
| Nível C — Definido | Métricas cobrem decisões/reutilização. |
| Nível B — Gerenciado Quantitativamente | Baselines e modelos quantitativos. |
| Nível A — Em Otimização | Métricas direcionam a melhoria. |
