# 🚀 Área de Processo — Melhoria Contínua

> Esta pasta é **autossuficiente**: tudo o que você precisa para entender a área está aqui.

## 📌 O que é esta área no MPS.BR
Melhoria e inovação contínuas do processo por análise causal. Corresponde à **Gerência de Inovação e Implantação (GIM)**.

## 🎯 Qual problema ela resolve
Resolve a **estagnação**: processos que não evoluem ficam obsoletos.

## ✅ O que entra nesta área
- Análise de causa raiz
- Plano de melhoria
- Inovação de processos
- Oportunidades e histórico de melhorias
- Benchmark e retrospectivas

## 🚫 O que NÃO entra nesta área
- Controle estatístico (→ gerencia-quantitativa)
- Métricas brutas (→ medicao)
