# 📈 Área de Processo — Gerência Quantitativa

> Esta pasta é **autossuficiente**: tudo o que você precisa para entender a área está aqui.

## 📌 O que é esta área no MPS.BR
Gestão estatística do processo e do projeto. Corresponde à **Gerência Quantitativa do Projeto (GQP)** e **Desempenho do Processo Organizacional (DEP)**.

## 🎯 Qual problema ela resolve
Resolve **não saber se as metas são estatisticamente atingíveis**.

## ✅ O que entra nesta área
- Objetivos quantitativos
- Baselines de desempenho
- Controle estatístico de processo
- Modelos de previsão

## 🚫 O que NÃO entra nesta área
- Coleta bruta de métricas (→ medicao)
- Melhoria de processo (→ melhoria-continua)
