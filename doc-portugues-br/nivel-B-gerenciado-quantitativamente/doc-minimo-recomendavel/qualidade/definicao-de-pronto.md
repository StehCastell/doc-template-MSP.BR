# Definição de Pronto (Definition of Done)

## Objetivo

Estabelecer critérios mínimos para considerar uma atividade concluída.

---

## Critérios

- [ ] Requisitos implementados.
- [ ] Código revisado.
- [ ] Testes executados com sucesso.
- [ ] Defeitos críticos resolvidos.
- [ ] Documentação atualizada.
- [ ] Alterações versionadas.

---

## Exceções

Descrever situações em que os critérios poderão ser flexibilizados.
