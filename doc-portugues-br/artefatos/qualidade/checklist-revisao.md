# Checklist de Revisão Geral

## Objetivo

Fornecer uma lista geral de verificação para revisões de produtos de trabalho.

---

## Documentação

- [ ] Documentos atualizados.
- [ ] Informações consistentes.
- [ ] Evidências registradas.

---

## Produto

- [ ] Critérios de aceitação atendidos.
- [ ] Defeitos críticos resolvidos.
- [ ] Requisitos implementados corretamente.

---

## Processo

- [ ] Procedimentos definidos foram seguidos.
- [ ] Checklists específicos foram executados.
- [ ] Aprovações necessárias foram obtidas.

---

## Configuração

- [ ] Itens versionados corretamente.
- [ ] Baselines atualizadas.
- [ ] Alterações registradas.

---

## Resultado da Revisão

- [ ] Aprovado
- [ ] Aprovado com ressalvas
- [ ] Reprovado
